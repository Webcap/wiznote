# Product Hunt Status Update

## Rich Text Editor Rolling Out 🎨

Hey WizNote community!

We're excited to announce that our **Rich Text Editor** is now gradually rolling out! 📝✨

### What's New

We've completely reimagined how you create and edit notes. WizNote now features a powerful markdown-style rich text editor that lets you:

- **Format your notes** with headers, bold, italic, and underline
- **Create structured content** with lists and code blocks
- **Add clickable links** to make your notes interactive
- **See your formatting in real-time** with our beautiful HTML viewer

### Gradual Rollout

As with all major features, we're rolling this out gradually to ensure a smooth experience. You'll start seeing the new editor if you're:

- ✅ A Premium member
- ✅ An early adopter who's already opted in via settings
- ✅ Part of our beta testing group

Don't worry if you don't see it yet! We're expanding access every day to ensure everything is working perfectly.

### How to Access

If you have access, you'll see the new editor automatically when creating or editing notes. The markdown syntax is intuitive and easy to learn:

- `# Header` for big titles
- `**bold**` for emphasis
- `- list` for bullet points
- `[link](url)` for clickable links

Your feedback helps us make WizNote even better! 🚀
