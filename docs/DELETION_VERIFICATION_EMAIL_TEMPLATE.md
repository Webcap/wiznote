# Account Deletion Verification - Email Templates

## Template 1: Initial Verification Request

**Subject:** WizNote Account Deletion Request Verification - Action Required

**Body:**

```
Hello,

We received a request to delete your WizNote account associated with this email address.

Request Details:
• Email: [USER_EMAIL]
• Ticket ID: [TICKET_ID]
• Submitted: [SUBMISSION_DATE]

⚠️ IMPORTANT: To confirm this request came from you, please reply to this email with the following statement:

"I confirm I want to delete my WizNote account and all associated data."

What will be deleted:
• Your account profile and settings
• All notes and documents (text, audio, PDF)
• All flashcards and quizzes
• All preferences and usage history
• Any active subscriptions will be cancelled

⚠️ This action cannot be undone.

If you did NOT make this request:
Please reply immediately so we can investigate and secure your account.

Processing Timeline:
We will process your deletion within 24-48 hours after receiving your confirmation.

Thank you,
WizNote Support Team
support@wiznote.app
```

## Support Agent Notes

### When to Use Each Template:

**Template 1:** First contact for every deletion request  
**Template 2:** User didn't respond after 48 hours  
**Template 3:** User confirmed, before starting deletion  
**Template 4:** After deletion is complete  
**Template 5:** User cancelled their request  
**Template 6:** Suspicious request or security concern  

### Response Time Guidelines:

- **Verification email**: Send within 4 hours of request
- **Follow-up**: Send after 48 hours of no response
- **Deletion**: Complete within 24-48 hours of confirmation
- **Completion email**: Send immediately after deletion

### Email Best Practices:

✅ Always be professional and empathetic  
✅ Use clear, simple language  
✅ Include ticket ID for tracking  
✅ Explain what will happen  
✅ Provide clear next steps  
✅ Give user option to cancel  

❌ Never rush the user  
❌ Don't make them feel bad for leaving  
❌ Avoid technical jargon  
❌ Don't promise specific timeframes you can't meet
